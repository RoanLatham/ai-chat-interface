Understanding Model Names

Llama model names can be confusing. Here's a quick guide:

  • Q Levels: (e.g., Q4_K_M vs Q5_K_M) - Higher Q numbers mean better quality but slower performance.
  • Parameter Count: ('M' million, 'B' billion) - higher parameter count results in a more knowledgeable but slower model.

Context Window Sizes

Larger context windows allow the model to remember more of the conversation history and handle more complex tasks.

Different Llama versions have different context window sizes:

  • Llama 2: 4,096 tokens
  • Llama 3: 8,192 tokens
  • Llama 3.1: 128,000 tokens
